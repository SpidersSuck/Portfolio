# Admin CMS Access Guide

## How to Access

1. **Click "Login"** in the top navigation bar
2. **Login** with the admin credentials:
   - Email: `admin@aww.com`
   - Password: `AWW2025!Admin`
3. **Look for "⚙️ Admin"** in the navigation menu after logging in
4. **Click "⚙️ Admin"** to access the Admin CMS

## Features

- ✅ Add new events with all details
- ✅ Upload event images
- ✅ Delete events
- ✅ View dashboard statistics

## Notes

- Events appear immediately on the public Events page
- Admin sessions last 24 hours
- First-time visitors see a helpful popup hint
